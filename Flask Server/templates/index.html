INTELLIGENT DRIVER SAFETY SYSTEM

CSE 705 Seminar Project - IOT

IOT application to avoid driver drowziness

This project seeks to increase traffic safety and offer drivers an intelligent safety system to prevent accidents.
It keeps the driver awake and prevents them from dozing off.
The system will track eye movement and determine if the driver's behavior indicates that the eyes have been closed for a prolonged period of time.
If this occurs, the system will automatically vibrate the steering and play a loud sound to notify the driver.
When the system determines that the vehicle's driver poses a serious risk,
The application will use the Telegram API to warn close family members about the driver's condition and inform emergency contacts.
